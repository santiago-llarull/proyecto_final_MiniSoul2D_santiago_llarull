Mini Souls 2D

INTENGRANTES: 
  santiago llarull 

Descripcion: 
  El presente proyecto propone un videojuego de combate táctico en 2D para PC, utilizando una perspectiva top-down (vista superior). El jugador controlará a un personaje dentro de una arena cerrada, con el objetivo principal de enfrentar y derrotar a un enemigo tipo Jefe (Boss). 
  La jugabilidad se centrará en la administración estratégica de recursos limitados, principalmente la vida y la estamina, obligando al jugador a decidir con precisión cuándo atacar, bloquear o esquivar. La experiencia está fuertemente inspirada en la filosofía de los juegos soulslike: cada error es penalizado de forma estricta, por lo que la victoria no depende de atacar sin parar, sino de la toma de decisiones bajo presión y el aprendizaje progresivo de los patrones del enemigo. 

Tecnlogias Utilizadas: 
  C#, MonoGame, .NET y visual studio

Como compilar y ejecutar:
  - Instalar Visual Studio asegurando marcar la opción de desarrollo de escritorio con .NET durante la instalación.
  - Instalar las extensiones y plantillas de MonoGame en Visual Studio para asegurar la compatibilidad.
  - Instalar Git en el sistema operativo para poder descargar los archivos.
  - Abrir una terminal de comandos en la carpeta específica de la computadora donde se desea guardar el proyecto.
  - Ejecutar el comando "git clone https://github.com/santiago-llarull/proyecto_final_MiniSoul2D_santiago_llarull.git" para descargar todo el código.
  - Entrar a la nueva carpeta del repositorio que se acaba de descargar.
  - Buscar y abrir el archivo de solución "Mini_Souls_2D.slnx" haciendo doble clic para que se cargue en Visual Studio.
  - Esperar unos segundos en la pantalla principal a que el programa restaure las dependencias de MonoGame de forma automática.
  - Revisar en el explorador de soluciones de la derecha que el proyecto principal esté configurado como proyecto de inicio.
  - Presionar la tecla "F5" o el botón de iniciar en la barra superior para compilar el código y abrir el juego.
